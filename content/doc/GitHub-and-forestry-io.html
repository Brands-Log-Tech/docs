If you import a git repository, Forestry will commit updates to your repository and watch it for changes.

When an editor saves changes in Forestry, 2 things happen:

  • All changes are committed and pushed to your GitHub repo
  • The site is compiled and sent to your host (FTP, S3, GitHub pages, etc)

When an dev pushes changes:

  • Forestry pulls in updates.
